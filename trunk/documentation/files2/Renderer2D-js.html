Renderer2D

srcRectangle,

color,
origin,
rotation)

call this in between begin and end calls to draw a Texture2D object to the screen

Parameters

texturea Texture2D object
srcRectanglea 4 element array containing [x position, y position, width, height] from the original texture source
destRectanglea 4 element array containing [x position, y position, width, height] determining where on the screen to draw the image
colora 4 element array containing [red, green, blue, alpha]
originthe pivot point or origin of the image.  the x and y positions for destRectangle start from the origin
rotationa float value determing the rotation of an image
